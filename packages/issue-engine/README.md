# @openforge/issue-engine

Estimates issue sizes, difficulty levels, and required effort for open-source project issues.

## Size Estimation
- **T-Shirt Sizes**: Maps issues to XS, S, M, L, XL sizes based on code volume, age, and label heuristics.
- **Estimated Effort**: Provides optional numerical estimations of effort values.
- **Confidence Rating**: Evaluates assessment reliability ratings.
