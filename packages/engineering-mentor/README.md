# @openforge/engineering-mentor

Educational orchestrator guiding open-source project contributors on solving issues step-by-step.

## Features
- **Socratic Concept Exploration**: Explains code patterns conceptually without generating code patches.
- **Dynamic Reading List**: Renders chronological files reading order using DAG dependencies.
- **Strict Guidelines & Retries**: Scans LLM results for code blocks or coding syntax, triggering validation retries if rules are broken.
