# @openforge/github-client

Provides a high-performance, strongly typed client for executing GraphQL and REST queries against GitHub.

## Features
- **Strong Typings**: Includes full Zod schema definitions for issues, repositories, commits, and comments.
- **Cursor-based Pagination**: Enforces robust cursor traversal on issue and repository queries.
- **Unified client**: Encapsulates GraphQL execution with client-side error handling and rate-limit tracking.
